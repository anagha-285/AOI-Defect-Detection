
# AOI Defect Detection using Deep Learning

This project implements an Automated Optical Inspection (AOI) system using deep learning to detect defects in images. It aims to improve quality control in manufacturing by automating defect detection.

---

## 🚀 Features
- Image preprocessing and augmentation
- Deep learning-based defect classification
- Model training and evaluation
- Visualization of predictions

---

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

---

## 📊 Results
- Achieved high accuracy in defect detection (update with your %)
- Model successfully distinguishes defective vs non-defective samples
